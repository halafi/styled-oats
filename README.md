## Getting Started

```sh
yarn add styled-oats
```

```jsx
import React from 'react'
import { Line } from 'styled-oats'

const BigLineComponent = () => (
    <Line fontSize={36} fontWeight={700}>
      BOIS
    </Line>
)
```
